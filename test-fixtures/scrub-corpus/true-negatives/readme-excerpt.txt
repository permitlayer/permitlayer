# My Project

A fast, lightweight web framework for building APIs.

## Getting Started

```bash
cargo install my-project
my-project init
my-project serve --port 8080
```

## Features

- Zero-copy request parsing
- Async/await native
- Built-in middleware stack
- OpenAPI 3.0 generation
- <1ms p95 latency on 100KB payloads
